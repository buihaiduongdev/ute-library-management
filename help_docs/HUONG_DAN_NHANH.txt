================================================================================
⚡ CÁCH TẠO FILE CHM - NHANH NHẤT
================================================================================

BƯỚC 1: Cài đặt HTML Help Workshop
--------------------------------------
1. Download từ: https://www.microsoft.com/en-us/download/details.aspx?id=21138
2. Chạy file cài đặt
3. Cài đặt với cấu hình mặc định

BƯỚC 2: Biên dịch file CHM
--------------------------------------
Cách 1 - Sử dụng script tự động (ĐƠN GIẢN NHẤT):
   • Double-click vào file: build.bat
   • Đợi biên dịch xong
   • File CHM sẽ xuất hiện: QuanLyDocGia_GiaHanThe.chm

Cách 2 - Sử dụng HTML Help Workshop:
   • Mở HTML Help Workshop
   • File > Open > Chọn "help_project.hhp"
   • Click nút "Compile" hoặc nhấn F9
   • Đợi biên dịch xong

Cách 3 - Sử dụng Command Line:
   • Mở Command Prompt
   • cd đến thư mục help_docs
   • Chạy lệnh:
     "C:\Program Files (x86)\HTML Help Workshop\hhc.exe" help_project.hhp

BƯỚC 3: Mở file CHM
--------------------------------------
1. Double-click file: QuanLyDocGia_GiaHanThe.chm
2. Nếu bị chặn (Windows Security):
   - Chuột phải vào file CHM
   - Properties
   - Tích "Unblock"
   - Apply > OK
   - Mở lại file

================================================================================
🔧 XỬ LÝ LỖI THƯỜNG GẶP
================================================================================

❌ Lỗi: "hhc.exe not found"
   ✅ Giải pháp: Cài đặt HTML Help Workshop hoặc sửa đường dẫn trong build.bat

❌ Lỗi: "Cannot compile"
   ✅ Giải pháp: 
      - Kiểm tra tất cả file HTML đều có trong thư mục
      - Đảm bảo file help_project.hhp và help_toc.hhc không bị lỗi

❌ Lỗi: File CHM bị chặn khi mở
   ✅ Giải pháp: Unblock file (xem Bước 3 ở trên)

❌ Lỗi: Hiển thị lỗi font hoặc tiếng Việt
   ✅ Giải pháo: 
      - Đảm bảo file được save với encoding UTF-8
      - Mở lại file CHM

❌ File CHM trống hoặc không có nội dung
   ✅ Giải pháp:
      - Kiểm tra file style.css có cùng thư mục không
      - Biên dịch lại
