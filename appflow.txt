[User Browser]
     │
     ▼
[index.html / pages/*.html]       ← Giao diện người dùng
     │
     ▼
[pages/*.js]                     ← Logic frontend theo role (admin.js, staff.js, user.js, auth.js)
     │
     │   (gọi API)
     ▼
[api/*.js]                        ← Các hàm fetch tới server (/api/books, /api/readers...)
     │
     ▼
[Server Routes]                    ← server/routes/*.js
  ├─ /api/auth (login, register, logout)
  ├─ /api/books
  ├─ /api/readers
  ├─ /api/borrow
  ├─ /api/return
  └─ /api/staffs
     │
     │   (middleware kiểm tra auth/role)
     ▼
[Middleware]                      ← server/middleware/auth.js
     │
     ▼
[Models / DB]                      ← server/models/db.js
     │
     ▼
[Database]                         ← SQL Server (bảng Books, Readers, Borrow, Users…)
